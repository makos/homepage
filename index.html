About me

My name is Mateusz, I'm a hobbyist programmer and web developer from Poland. At the moment I'm an amateur but I'm working hard to make this my living either at a company or by freelance work. Currently I'm employed as a CAD designer / junior manager at a small private company producing steel and aluminium parts.

I'm also an avid gamer. I've been playing games ever since I can remember, and it's what sparked my love for computers. Around 14 I got really interested in programming and since then I've amassed experience in using a vast array of operating systems (many versions of Windows and all big Linux distributions along with some more niche like Arch and Gentoo) and programming languages (I can understand fairly simple C/C++ code, read and write more advanced C# code thanks to the Unity Engine which I used in my gamedev voyages and am fluent in scripting languages like Python and JavaScript).

Currently I'm going through the freeCodeCamp curriculum, as seen in my projects.
